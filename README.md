slow-api
--------

Serve for tailing slow sql log, slow http api, and alerts via email, slack ...

## Features

- collect and store

  - slow sql
  - slow http api

- admin UI

- alert

  - email
  - slack

## USE

- Run as serve

`slow-api serve`

- Import log 

`slow-api import -f /home/koolay/slow_sql.log`
